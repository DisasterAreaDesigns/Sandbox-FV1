Format Information

  • Each input file should contain exactly 129 lines
  • Files are mapped to consecutive 512-byte blocks (0000-01FF, 0200-03FF, etc.)
  • Output is always 1025 lines with unused areas padded (last byte of each line = 0x11)
  • Drag files to reorder them in the final EEPROM image
